MisinfoX is an AI-powered Chrome browser extension designed to help users identify misinformation, phishing attempts, and misleading online content in real time. It allows users to verify suspicious text directly on any website by highlighting content and analyzing it instantly, without leaving the page.

Key Features

Highlight and Verify
Select any text on a webpage and analyze it instantly using the MisinfoX extension.

Real-Time AI Analysis
Detects misleading claims, phishing attempts, and suspicious content while browsing.

Clear Verdicts
Provides easy-to-understand results such as Safe, Scam, Suspicious, or Unverified, along with brief explanations.

Inline Results Display
Analysis results appear directly on the webpage in a floating card for a seamless experience.

Privacy-Focused Design
Only the selected text is processed. No personal data or browsing history is collected or stored.

Dark and Light Mode Support
Automatically adapts to the browser theme for better usability.

Purpose

Online misinformation spreads rapidly across websites and digital platforms. MisinfoX enables users to verify content before trusting or sharing it, helping promote informed decision-making and safer browsing.

How It Works

Highlight any text on a webpage

Click the "Ask MisinfoX" option

The AI analyzes the selected content

View the verification result instantly on the same page

Technology Stack

Frontend: HTML, CSS, JavaScript

Browser Extension: Chrome Extension (Manifest V3)

Backend: Node.js-based analysis service

Architecture: Content Scripts and Background Service Worker
